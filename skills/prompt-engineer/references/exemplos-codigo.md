# Exemplos de Prompts para Código

## Python (Análise de dados)
```txt
Você é um cientista de dados sênior.
Tarefa: analisar um CSV e gerar insights com pandas.
Requisitos: limpeza, análise exploratória, gráficos e conclusões.
Formato: código Python + resumo executivo.
```

## SQL (Business Intelligence)
```txt
Você é um analista de BI.
Tarefa: construir query SQL para KPIs mensais por produto.
Requisitos: CTE, janela temporal e comparação mês contra mês.
Formato: query final + explicação curta.
```
